Caçador
Catedral São Francisco de Assis

Com uma área de 983,424 km² [4], Caçador está localizada no Alto-Vale do Rio do Peixe, meio-oeste de Santa Catarina, integrando a região Metropolitana do Contestado.[1] O território do município limita-se ao norte com Calmon e General Carneiro (PR), ao sul com Rio das Antas e Videira, ao leste com Lebon Régis e ao oeste com Água Doce e Macieira.

Sua altitude média é de 1 000 m, estando o ponto culminante do território municipal a 1 390 m de altitude (Elevação de Rio Verde) e o ponto mais baixo a 780 m acima do nível do mar.[18]

HIDROGRAFIA

O território do município é rico em recursos hidro-minerais, situando-se em sua totalidade sobre o Aquífero Guarani. É banhado por vários rios, dentre os principais o que deu o nome à cidade, Caçador, e os do Peixe, Castelhano, XV de Novembro, Jangada, Preto, São Pedro e Veado.

CLIMA

A cidade possui clima temperado subtropical úmido (Classificação climática de Köppen-Geiger Cfb).[3] Entre 1977 e 2004 foram registrados temperatura média anual de 16,3 °C e precipitação acumulada média anual de 1716mm.[19]

Os verões são quentes e úmidos, com máximas de temperatura e precipitação em janeiro. Os invernos são frios, menos úmidos que os verões, alternando períodos chuvosos (mês de junho) e secos (meses de julho e agosto). O frio é mais intenso durante os meses junho e julho e nos períodos secos do inverno, o clima, apesar do frio, é considerado agradável. Nas últimas duas semanas do mês de maio ocorre o fenômeno conhecido popularmente como "Veranito de Maio", caracterizado por repentina elevação das temperaturas em pleno outono.

Em Caçador registrou-se oficialmente a menor temperatura já ocorrida no território brasileiro: −14 °C, em 11 de junho de 1952.[22] Outros registros oficiais incluem temperaturas mínimas extremas de −11 °C e de −10,4 °C, respectivamente 10 de julho de 1952 e de 6 de agosto de 1963.[21][22][23]

Geadas ocorrem frequentemente de abril a setembro, com 1271 ocorrências registradas entre 1942 e 2006. A queda de neve é mais rara, com apenas 20 ocorrências registradas durante o mesmo período.[21] A última ocorrência sendo em julho de 2013.[24]

Exercício 4

CSS Inline: É quando você personaliza diretamente no componente, na linha que será usado. É util por exemplo se você quiser fazer alguma mudança mais especifíca naquele local, porém para ser usado muitas vezes não faz muito sentido pois acaba sendo muito repetitivo.

CSS Interno: É quando você faz as personalizações no head do seu código HTML, sendo usando uma classe, um ID ou por componente. É util pois você consegue manter organizado as suas edições no head, além de conseguir fazer customizações que podem ser usadas varias vezes porém escrevendo o código só uma vez economizando tempo e código.

CSS Externo: É modo mais organizado, pois você faz seu código CSS separadamente em outro arquivo, assim não misturando com o código HTML e deixando um arquivo mais limpo. É um metodo que tende a deixar projetos mais sofisticados, porém, muitas importações CSS podem deixar sua página pesada demais, e por exemplo acabar demorando para aplicar as personalizações, deixando o site "cru".

Exercício 5

Seletor CSS Classe: A classe é usada para fazer personalizações que podem ser usadas em varios elementos. Por exemplo uma classe ".Azul" que deixa o background na cor azul, se você quiser que um grupo de elementos como um h1 e uma table sejam azuis, precisará apenas adicionar a classe azul a eles.

Seletor CSS ID: A ID é usada para fazer personalizações em um determinado elemento. Por exemplo uma ID "H1" vai conter customizações caracteristicas dos elementos h1, e apenas desses elementos, padronizando a estética deles.

Seletor CSS Elemento: A Personalização em Elemento é usada para customizar TODOS os componentes especificados nela da tela. Por exemplo, ao customizar um componente "h1", todos os usados na tela terão a mesma customização, sem precisar usar classes ou IDS.

Elemento CSS Border-radius: Adiciona cantos arredondados as bordas.

Elemento CSS animation: Permite colocar colocar uma animação a um elemento DIV.

Elemento CSS visibility: Permite controlar se os elementos serão visiveis ou invisiveis.

Elemento CSS word-spacing: Define o espaço das palavras dentro de um texto.

Elemento CSS transition: Faz a transição de um elemento em alguma aão ou transformação que o mesmo faça.
